# A&H ASESOL - Estrategia Financiera Integral

![Next.js](https://img.shields.io/badge/Next.js-15-black?style=for-the-badge&logo=next.js)
![Tailwind CSS](https://img.shields.io/badge/Tailwind_CSS-3.4-38B2AC?style=for-the-badge&logo=tailwind-css)
![TypeScript](https://img.shields.io/badge/TypeScript-5-blue?style=for-the-badge&logo=typescript)

Plataforma corporativa de alto rendimiento diseñada para **A&H ASESOL**, una firma líder en consultoría contable, financiera y auditoría. Este sistema optimiza la presencia digital de la marca mediante una arquitectura moderna, segura y orientada a la conversión.

---

## 🛠️ Tecnologías de Implementación

- **Framework:** [Next.js 15+](https://nextjs.org/) (App Router Architecture).
- **Estilizado:** [Tailwind CSS](https://tailwindcss.com/) para una UI fluida y responsive.
- **Seguridad:** Implementación de *Security Headers* (XSS, Clickjacking protection).
- **Contacto:** Integración robusta con Formspree para gestión de leads.
- **Optimización:** SEO técnico avanzado y protección de código fuente en cliente.

## 🚀 Características Clave

* **Diseño Corporativo:** Interfaz minimalista con estética de alta gama (Modo Dark/Light adaptado).
* **Seguridad Activa:** Bloqueos de inspección de código y protección de clic derecho para integridad de marca.
* **Performance:** Carga optimizada mediante renderizado híbrido.
* **Brochure Digital:** Sección dedicada a la exposición detallada de servicios financieros.
